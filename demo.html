Mast CSS Framework

Mast is a CSS framework designed to remove the mess of designing informative web pages. It is intended for experienced HTML writers as a foundation to write informative and simplistic web pages. The core idea behind Mast is focus on content, not design. After all, a good design is one you never notice.

The following line is not a simple HR horizontal rule element, but a styled DIV with the class name "hr". It is much easier to deal with than the usual horizontal rules.

This page demonstrates some of the uses of the MAST CSS system and its various styles. It shows a 900px wide main DIV that contains text and other elements such as bold text. The Mast CSS itself can be included with the following stylesheet statement:

<link rel="stylesheet" type="text/css" href="mast/mast.css" />

Other common styles include the inserted text, deleted text, and abbreviations. Code snippets are wrapped by the HTML5 <code> tag. They are produced in monospace on a block element and have a special border that indicate that they are code.

function dosomething(){
    //Occasionally, wrapping code in a <pre> tag helps to improve its readability
    return "<code><pre>Hello world!</pre></code>";
}
